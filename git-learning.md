## 📚 Что именно изучено

### Основы
- Установка и настройка Git
- Инициализация репозитория (`git init`)
- Основные команды: `status`, `add`, `commit`, `log`, `diff`
- Работа с удалёнными репозиториями (`clone`, `push`, `pull`, `fetch`)

### Ветвление и слияние
- Создание, переключение и удаление веток (`branch`, `checkout`, `switch`)
- Слияния: `merge`
- **Rebase** как альтернатива слиянию (интерактивный режим)
- Разрешение конфликтов (ручное и с помощью инструментов)

### Продвинутые темы (на базе тренажёра)
- `cherry-pick` – перенос конкретных коммитов
- Относительные ссылки (`^`, `~`)
- Игнорирование файлов (`.gitignore`)
- Работа с тегами (`tag`)

### Совместная работа
- Работа с pull request'ами на GitHub/GitLab
- Стратегии ветвления (Git Flow, GitHub Flow)
- Синхронизация форков (upstream, fetch + merge)

**🎯 Итог:**  
Теперь я уверенно ориентируюсь в графах коммитов, умею переписывать историю без страха, чинить конфликты и эффективно работать в команде. Весь материал подкреплён практикой на визуальном тренажёре – скриншоты выше это подтверждают.
