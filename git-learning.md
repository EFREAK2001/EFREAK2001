# Моё изучение Git

## Основы
- Установка и настройка Git
- Инициализация репозитория (`git init`)
- Основные команды: `status`, `add`, `commit`, `log`, `diff`
- Работа с удалёнными репозиториями (`clone`, `push`, `pull`, `fetch`)

## Ветвление и слияние
- Создание, переключение и удаление веток (`branch`, `checkout`, `switch`)
- Стратегии слияния (`merge`, `rebase`)
- Разрешение конфликтов

## Продвинутые темы
- Интерактивный rebase
- Использование `git stash`
- Работа с тегами (`tag`)
- Игнорирование файлов (`.gitignore`)

## Совместная работа
- Работа с pull request'ами на GitHub/GitLab
- Стратегии ветвления (Git Flow, GitHub Flow)

## Практические проекты
- Настройка CI/CD через GitHub Actions (триггеры на push/pull request)
- Работа с Git в команде (peer review, синхронизация форков)

**Итог:** Уверенное владение Git для повседневных задач и командной разработки.
